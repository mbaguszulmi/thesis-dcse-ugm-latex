# Thesis Template in Latex for DCSE UGM

Cloned from [https://github.com/0x0v3rlo4d/thesis_latex-dcse_ugm](https://github.com/0x0v3rlo4d/thesis_latex-dcse_ugm)

- LaTeX thesis template for DCSE UGM
- Update fonts
- Designed for Firebase Studio (Google IDX) and Codespace

## Firebase Studio Setup

1. Fork this Repo
2. Copy the url of your newly created repo
3. Go to [https://studio.firebase.google.com/](https://studio.firebase.google.com/)
4. Click Import Repo
5. Paste Repo URL
6. Complete the rest of the setup

> Warning: The Firebase Studio will be sunset on March 22, 2027. It's recommended to use Codespace instead.

## Codespace Setup

1. Fork this Repo
2. Go to https://github.com/codespaces/new
3. Select the new repo
4. Complete the rest of the setup

> Pro Tip: you can run this remotely from local VSCode. Open Hamburger menu (Top Left corner) > Open in VS Code Desktop.
